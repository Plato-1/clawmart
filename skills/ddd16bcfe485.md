# One-Click Deploy
## Usage
deploy --target railway --domain agent.yours.com
## Supported Targets
- Railway (easiest, $5/mo)
- Fly.io (global edge, $6/mo)
- Vercel (serverless, free tier)
- Self-host (your VPS)
## Includes
Auto-Dockerfile generation, HTTPS via Caddy, health checks, restart-on-failure, environment variable management
