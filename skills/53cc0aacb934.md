# A2A Payment Bridge MCP Server

Agent-to-agent payments via smart accounts with delegated permissions. Agents pay each other for services, data, compute without human approval per tx. ETH/USDC/USDT on Base and Ethereum. Based on x402 + Google A2A standards.

## Usage
Install via ClawMart. Works with Claude Code, Hermes Agent, Cursor.
